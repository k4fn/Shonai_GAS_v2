# Shonai\_GAS\_v2 使い方ガイド

折衝表のGoogleスプレッドシートから、自分の団体が利用する部屋・日時を読み取り、学内集会願へ貼り付けるためのスプレッドシートを自動作成するGoogle Apps Scriptです。

現行版は、新しい学内集会願フォーマット（`【03】学内集会願.xlsm` 相当）を基準にしています。

## 作成されるもの

* 部屋ごとの学内集会願コピペ用シート

* 1シートあたり最大15件の利用予定

* サークルメンバーへの共有に使える `周知用` シート

<img width="439" alt="学内集会願の出力例" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/59597a90-e379-4ccb-a3db-556d06317318">
<img width="180" alt="周知用シートの出力例" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/3e4bbeea-9d6f-4cf6-9736-a04499c5f948">

出力シートでは、以下の列に自動で値が入ります。

| 内容                   | 出力先  |
| -------------------- | ---- |
| 施設名                  | A列   |
| 月/日(曜)               | C列   |
| 1H〜放課後の利用マーク         | E〜L列 |
| 利用開始時刻               | N・P列 |
| 利用終了時刻               | R・T列 |
| 人数                   | U列   |
| 同日に複数の利用時間がある場合の追加時間 | V列   |

## 注意事項

* 自己責任で使用してください。

* 提出前に、施設名・日時・曜日・人数・備考欄を必ず確認してください。

* 1月分を作る場合は、折衝表の日付の年が正しいか確認してください。曜日ずれの原因になります。

* `テンプレ.xlsx` は、そのままのExcelファイルURLでは使えません。Google Driveにアップロード後、Googleスプレッドシート形式に変換したもののURLまたはIDを使ってください。

## 使い方

### 1. Google Apps Scriptを作成

1. [Google Apps Script](https://script.google.com/home/start) を開く。
2. 新しいプロジェクトを作成する。
3. [コード.gs](/コード.gs) の内容をコピーして貼り付ける。
4. プロジェクトを保存する。

### 2. テンプレートを用意

1. [テンプレ.xlsx](/テンプレ.xlsx) をダウンロードする。
2. Google Driveにアップロードする。
3. アップロードしたファイルをGoogleスプレッドシートで開く。
4. Googleスプレッドシート形式として保存する。
5. 保存したGoogleスプレッドシートのURLまたはIDを控える。

<img width="500" alt="テンプレートをGoogleスプレッドシートとして保存する例" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/bd1b4202-95c2-4874-9fff-8653b1f5f1b6">

テンプレート内のコピー元シート名は `テンプレ` を想定しています。別名の場合でも、最初のシートをコピー元として使います。

### 3. コードの初期設定

`コード.gs` の上部にある設定値を編集します。

| 定数名                         | 内容                                     |
| --------------------------- | -------------------------------------- |
| `NEGOTIATION_SHEET_URL`     | 折衝表のGoogleスプレッドシートURLまたはID             |
| `CIRCLE_NAME`               | 折衝表で検索する団体名                            |
| `NEW_SPREADSHEET_BASE_NAME` | 作成されるファイル名の先頭部分                        |
| `TEMPLATE_SPREADSHEET_URL`  | Googleスプレッドシート形式に変換したテンプレートのURLまたはID   |
| `OUTPUT_FOLDER_ID`          | 出力先Google DriveフォルダID。マイドライブ直下でよければ空文字 |
| `ROOMS`                     | 処理対象にする部屋名                             |
| `NUMBER_OF_USERS`           | 学内集会願に記入する人数                           |

<img width="500" alt="コード上部の初期設定例" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/639ed0e0-f0cb-4579-a501-d830bba2e0e0">

2回目以降は、基本的に折衝表URLやテンプレURLなど必要な箇所だけ変更すれば使えます。

## 実行方法

1. Apps Script画面で実行する関数に `createMeetingRequestSpreadsheets` を選ぶ。
2. 「実行」ボタンをクリックする。
3. 初回のみ権限確認を行う。
4. 実行ログに出たURL、または指定した保存フォルダから作成されたスプレッドシートを開く。

<img width="500" alt="Apps Scriptの実行ボタン" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/43e84635-1cc1-4b16-8897-6c6f9470fa07">

初回の権限確認では、Googleアカウントを選び、詳細表示からプロジェクトへ移動して許可してください。2回目以降は通常スキップされます。

## ファイルの取得とコピペ

1. 作成されたスプレッドシートを開く。
2. `ファイル` → `ダウンロード` → `Microsoft Excel（.xlsx）` でダウンロードする。
3. ダウンロードした `.xlsx` をExcelで開く。
4. 必要な行を新しい学内集会願へ貼り付ける。

<img width="500" alt="xlsx形式でダウンロードする例" src="https://github.com/k4fn/Shonai_GAS/assets/121530063/95897698-efa8-4199-ba24-dedc8995ac60">

Googleスプレッドシートから直接コピペすると、書式が崩れる場合があります。Excel形式でダウンロードしてから貼り付ける運用を推奨します。

## よくあるエラー

### `Invalid argument: url`

`TEMPLATE_SPREADSHEET_URL` または `NEGOTIATION_SHEET_URL` に、GoogleスプレッドシートではないURLが入っている可能性があります。

* `.xlsx` のDrive URLではなく、Googleスプレッドシート形式に変換したファイルのURLまたはIDを指定してください。

* URLの `/spreadsheets/d/` の後ろにある長い文字列がスプレッドシートIDです。

* アクセス権がないファイルも開けません。

### テンプレートシートが見つからない

テンプレートのスプレッドシートにシートが存在するか確認してください。`テンプレ` というシートがあればそれが使われます。ない場合は最初のシートが使われます。

### 出力に不要な部屋が含まれる

`ROOMS` から不要な部屋名を削除してください。処理対象が減るので実行も少し速くなります。
